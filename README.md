# Safe Pointing - Standalone Pointing System for RedM

A high-performance, **STANDALONE** script for RedM that allows players to point where they are looking. This resource features smooth network synchronization and intelligent camera-based raycasting to enhance roleplay immersion.

## 🌟 Key Features
* **Standalone:** Compatible with ALL frameworks like VORP, RSG, RedEM:RP, QBR, or Vanilla servers.
* **Precise Raycasting:** Uses camera direction and raycasting to ensure the character points exactly where the player is looking.
* **Full Synchronization:** Seamlessly updates for all players on the server via optimized network events.
* **Performance Optimized:** Written in Lua 5.4 with low CPU/Resmon footprint.

## 🛠️ Installation
1. Download the repository.
2. Extract the `safe_pointing` folder into your server's `resources` directory.
3. Add `ensure safe_pointing` to your `server.cfg`.

## 🎮 How to Use
* Press **B** (Default Key: `0x4CC0E2FE`) while on foot to start or stop pointing.
